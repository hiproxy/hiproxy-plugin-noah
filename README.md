# hiproxy-plugin-noah

hiproxy Noah plugin, can only be used on our **internal network**.

## Installation

* install hiproxy

```bash
npm install -g hiproxy
```

* install noah plugin

```bash
npm install -g hiproxy-plugin-noah
```

## Usage

```bash
# 26572 is the envid
hiproxy noah 26572
```
